# Проект тестирования Codebattle

**Тестируемое приложение:** [Codebattle](https://codebattle.hexlet.io) — игровая платформа для программистов с открытым исходным кодом, разрабатываемая сообществом Хекслет.

## Содержание файлов

- `requirements.yml` — функциональные, нефункциональные и неявные требования к проекту
- `test-cases.yml` — подробные тест-кейсы для проверки функциональности
- `testing-report.yml` — результаты выполнения тест-кейсов (пройдены/провалены)
- `issues.yml` — ссылки на открытые баг-репорты в GitHub Issues
- `reopened_issues.yml` — ссылки на переоткрытые баги (неисправленные после закрытия)

## О проекте Codebattle

Codebattle — это игра, в которой два программиста одновременно решают одну задачу на разных языках программирования. Кто выполнил быстрее — тот побеждает.

**Основные возможности:**
- Соревнования с реальными игроками, ботами или друзьями
- Поддержка нескольких языков программирования
- Чат во время игры
- Рейтинги и статистика игроков
- Просмотр записей завершенных игр

## Цель тестирования

Проверка функциональности, производительности и удобства использования платформы Codebattle, а также выявление критических дефектов
